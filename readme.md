# Personal Dynamic Domain Name Service Sync Script (PDDNSSS)

[![PHP Version](https://img.shields.io/badge/PHP-8.0%2B-blue.svg)](https://www.php.net/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

A PHP script that automatically updates DNS records and database whitelists when your public IP address changes. Perfect
for home servers or development environments with dynamic IP addresses.
